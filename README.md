## 📌 Project Overview
The **AI Meeting Summarizer** is a web application that allows users to upload or paste meeting transcripts and generate AI-powered summaries with key discussion points, decisions, and action items.

---

## ✨ Features

- 📝 Upload or paste meeting transcripts
- ⚡ AI-powered summary generation
- 📅 Extract key discussion points
- ✅ Action items with owners
- 📧 Share summaries through email
- 🎨 User-friendly interface
- 🚀 Built using modern web technologies

---

## 🛠 Tech Stack

- Framework: Next.js
- Styling: TailwindCSS + Shadcn UI
- Language: TypeScript
- AI API: Groq API
- Email API: Resend
- Deployment: Vercel
